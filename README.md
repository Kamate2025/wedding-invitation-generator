# 💍 Wedding Invitation Generator

A web-based system that generates personalized wedding invitation and contribution letters for multiple recipients. Built using HTML, CSS, and JavaScript (Canvas API).


## ✨ Features

- 🎯 Generate personalized invitations per recipient
- 🌍 Supports English and French language templates
- ➕ Add multiple recipients dynamically
- 🖼️ Uses HTML5 Canvas for image rendering
- ⬇️ Instant download of generated invitations
- 🔐 Simple password-protected access system
- 🎨 Clean and responsive UI design


## 🧠 How It Works

1. User logs into the system using a password
2. Adds one or more recipient names
3. Selects language (English or French)
4. Clicks **Generate Invitations**
5. System:
   - Loads template image
   - Writes recipient name on canvas
   - Generates and downloads personalized image

## 🛠️ Tech Stack

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (Vanilla JS)
- Canvas API
- Bootstrap Icons
- Google Fonts
